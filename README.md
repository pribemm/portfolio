# Gmail API com Jupyter Notebook

Este repositório contém um notebook Jupyter que demonstra como usar a **Gmail API** para acessar e gerenciar e-mails do Gmail de forma programática. 
O notebook cobre desde a configuração inicial até operações práticas como listar mensagens, ler conteúdos e baixar anexos.

## 📌 Sumário
- O que é uma API?
- Preparação do Ambiente
- Autenticação com script python
- Listar dados

## Pré-requisitos
Antes de começar, você precisará ter:
- Uma conta Google (Gmail).
- Python 3.7 ou superior instalado.
- Conhecimentos básicos de Python e Jupyter Notebook.
- Acesso à internet para instalar pacotes e autenticar.
