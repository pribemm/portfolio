# Portfólio

Este repositório documenta minha jornada de aprendizado e prática em projetos reais.

O que você encontra aqui:
- *Análises exploratórias* com visualizações claras.
- *Modelos preditivos* simples e didáticos.
- *Código* bem estruturado e explicado linha a linha.
- *Otimização* com a utilização de modelos matemáticos.

## Para recrutadores
Navegue pelos notebooks para avaliar minhas habilidades em: Python, Pandas, Matplotlib, Scikit-learn. 
Cada projeto inclui um README próprio com o contexto de negócio e os resultados alcançados.

## Para curiosos e iniciantes
Todos os projetos têm uma versão em HTML (sem código) disponível na pasta /html-versions. Basta abrir no navegador para ler como se fosse um artigo.
